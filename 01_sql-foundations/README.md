# SQL Foundations

## Objectif
Modélisation d'une base de données immobilière et requêtes business sur les états des lieux.

## Contenu
- `schema.sql` : création des tables
- `sample_data.sql` : données de démonstration
- `queries.sql` : requêtes d'analyse métier

## Comment utiliser
1. Ouvrir `schema.sql` et `sample_data.sql`
2. Copier les scripts dans un outil SQL (PostgreSQL ou SQLite)
3. Exécuter les requêtes dans `queries.sql`

## Résultat attendu
- Volume d'inspections par agence
- Défauts les plus fréquents par pièce
- Durée entre état d'entrée et de sortie
- Liste des biens à risque (défauts graves)

## Ce que ce module démontre
- Modélisation relationnelle
- Requêtes SQL multi-tables (JOIN, GROUP BY)
- Analyse orientée métier immobilier
